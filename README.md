## 🚀 GitHub Pages-এ প্রকাশ

1. GitHub Repository খুলুন।
2. `index.html`, `style.css`, `script.js` এবং `README.md` আপলোড করুন।
3. **Commit changes** করুন।
4. **Settings → Pages** এ যান।
5. **Source** হিসেবে **Deploy from a branch** নির্বাচন করুন।
6. **Branch: main** এবং **Folder: /(root)** নির্বাচন করে **Save** করুন।
7. কয়েক মিনিট পর আপনার ওয়েবসাইট লাইভ হয়ে যাবে।
